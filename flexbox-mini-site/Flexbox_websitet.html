Alignment and space

Flexbox allows us to align and distribute items along the main axis (horizontal) or cross axis (vertical). We can also control the spacing between items using the gap property.

Tricky Positioning

Flexbox can be tricky when dealing with tricky positioning scenarios, such as centering items horizontally or vertically. We can use the justify-content and align-items properties to achieve these effects.
